02_SPECIFICATIONS

FUZZY_SEARCH

utilizes a local title index for zero-latency searching across thousands of wiki entries.

TUI_ENGINE

custom mediawiki parser specifically designed for terminal-safe rendering of tables and callouts.

CLIPBOARD_INTEGRATION

direct extraction of code blocks from wiki pages to system clipboard using the c key.

LOCAL_CACHE

automatically persists visited pages to disk for offline survival during network outages.

KISS_PHILOSOPHY

zero bloat implementation. no branding, no telemetry, no unnecessary headers.

BUILT_WITH_GO

statically compiled binary for high performance and minimal system dependencies.
