# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0-beta.1] - 2026-05-26

### Added
- Initial beta release preparation.
- Standardized versioning using `VERSION` file and `ldflags`.
- Enhanced `Makefile` with release targets.
- Added `CONTRIBUTING.md` and initial `CHANGELOG.md`.
- Bubble Tea based terminal UI for Arch Wiki.
- Hybrid search (local index + API).
- Markdown rendering via Glamour.
- Offline mode support with local cache and archive.
- Code block extraction and clipboard integration.
